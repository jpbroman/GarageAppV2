This is the GarageExercise V2 project

It is written in C# and allow you to manage a garage with several vehicles of various types:

* Motorcycle
* Airplane
* Car
* Bus
* Boat

Enjoy a faboulus Console UI to park, unpart, show or search for
vehicles without having to go to the physical garage.

You can save an inventory list with all your vehicles in the garage.
Ofcourse you can also load an inventory list, either by supplying a
filename on the command line or via the application menu.

Command line example:
> GarageAppV2 garageV2.inv

File suffix can be what ever you want, but file format must adhere to:

Car, Volvo, Red, ABC133, 5, Sedan, Manual
Motorcycle, Klochi, Blue, ASD345, 1, Standard, FourStroke
Bus, Mihaya, Red, POI123, 40, City
Airplane, Flyer2, Blue, JKL987, 40, Commercial, 2
Boat, Floater21, White, BOA098, 8, Motorboat, 21

That is:
Vehicle type, Make, Color, Registration number, Number of seats, (vehicle specific data).

Enjoy!
